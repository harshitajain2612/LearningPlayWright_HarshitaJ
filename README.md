# Learning Playwright — Harshita

Simple Playwright examples and exercises collected in this repository.

Quick start

```bash
# install dependencies
npm install

# run Playwright tests
npx playwright install --with-deps
npm test
```

Repository layout

- Chapter_01_Basics/: basic Playwright examples
- Chapter_02_Java_Concepts/: JS fundamentals exercises
- Chapter_03_identifier_Literal_Operators_Statements/: identifiers, literals, operators
- Chapter_04_Operators/: assignment, comparison, logical, string, ternary, typeof operators
- Chapter_05_Statements/: if/else statements and conditional logic
- Chapter_06_Switch_Statements/: switch cases and default handling
- Chapter_07_Loops/: for, while, do-while loops
- Chapter_08_Arrays/: array creation, access, modification, add/remove elements
- Chapter_09_Functions/: function declarations and exercises
- Chapter_10_String/: string operations and methods
- Chapter_11_Objects/: object creation and manipulation
- Chapter_12_MultiDimensionalArray/: multi-dimensional array examples
- Chapter_13_Callback/: callback function examples
- Chapter_14_PROMISES/: promise handling and patterns
- Chapter_15_Async_Await/: async/await patterns and exercises
- Chapter_16_OOPS_concept/: object-oriented programming concepts
- Chapter_17_OOPS_Inheritance/: inheritance types (single, multi-level, multiple, hierarchical)
- Chapter_18_TypeScript/: TypeScript basics and type annotations
- chapter_19_Typescript_Interface/: interfaces, extends, readonly, optional properties, real-world QA examples
- Chapter_20_Typescript_Enums/: enums with string and numeric values
- Practice_Object_Classes/: practice exercises with classes and objects
- PracticeTasks/: practice tasks for interfaces and advanced TypeScript
- Lecture_Playwright_MCP/: Playwright with Model Context Protocol
- tests/: Playwright test files
- specs/: test specifications
